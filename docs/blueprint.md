# **App Name**: Synapse

## Core Features:

- Quiz Question Presentation: Display clinical laboratory science questions with multiple-choice options on a dedicated quiz screen.
- Instant Answer Feedback: Provide immediate validation and visual feedback to the user upon submitting each answer.
- Quiz HUD Management: Allow users to toggle the visibility of the quiz timer and item list (navigation sidebar) for focused concentration.
- Assessment Completion Celebration: Award passers with a 'Future RMT' badge and a visual confetti animation upon successfully completing an assessment.
- Admin Content Upload: An administrative portal for uploading module titles, descriptions, and direct download links (e.g., from Google Drive, Dropbox, GitHub) using Firestore.
- Student Study Module Access: Allow students to view available study materials, complete with titles, descriptions, and buttons to download or view associated PDFs from external links.
- Secure User Authentication: Implement Firebase Authentication for both student and teacher (admin) user logins, securing access to appropriate app functionalities.

## Style Guidelines:

- The primary accent color for interactive elements and highlights is a vibrant emerald green (#26A95A), conveying a sense of energy and sophistication against dark backgrounds.
- The background color is a deep, desaturated charcoal green (#212C26), forming the base of an ultra-sleek dark mode experience.
- A softer, light green (#B0E0B0) serves as an accent for secondary UI elements and subtle highlights, maintaining theme consistency while offering contrast.
- Headlines and prominent text will use 'Space Grotesk', a sans-serif font, for a modern and tech-inspired feel. Body text will utilize 'Inter', a clean and highly readable sans-serif font, ensuring clarity for clinical content.
- A dynamic pulsing 'N' or Synapse icon will be featured prominently on splash screens and login pages to establish brand identity.
- The Quiz Screen layout will prioritize readability and minimal distraction, featuring clearly separated question and answer areas with readily accessible HUD controls for toggles.
- Subtle, smooth animations, including a pulsing logo and celebratory confetti, will enhance user engagement and provide satisfying feedback.
